🛠️ Update Instructions
# Prepare Files

⚠️ Do NOT extract the 6315_1.zip file (or similarly named update package).


# Prepare USB / SD Card
1. Use a USB flash drive or SD card (max 128GB recommended).
2. Format it to FAT32.
3. Copy all extracted files directly to the root directory (not inside any folder).

# Install the Update
1. Insert the USB/SD card into the head unit.
2. Wait for the system to detect the update automatically.
3. Tap "Start" when prompted to begin the update.

⏳ Update Duration
* The update process takes approximately 10 minutes.

✅ Completion
* When the update is successful, the screen will display: update success! Pls remove device (12)
* Remove the USB/SD device.
* The unit will reboot automatically.

⚠️ Important Notes
* 🔋 Ensure stable power during the update (do not turn off the vehicle).
* 💾 Use a clean, empty USB/SD card.
* 🧹 Delete any existing files before copying the update files.
* ❌ Do NOT interrupt the update process — this may damage the device.
* 📁 Always copy files to the root directory, not inside folders.
